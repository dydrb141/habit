API Endpoints
RESTful API + WebSocket. 모든 엔드포인트는 /api/v1 프리픽스. JWT Bearer 인증.
MethodEndpoint설명인증
POST/auth/register회원가입 (부모/자녀)-
POST/auth/loginJWT 토큰 발급-
POST/auth/pair부모-자녀 페어링 (6자리 코드)Parent
GET/character내 캐릭터 상태 (스탯, HP, 장비)Child
PUT/character/equipment장비 장착/해제Child
POST/character/shop/buy골드/보석으로 아이템 구매Child
POST/habits습관 생성 (부모 or 자녀)Both
GET/habits/today오늘의 퀘스트 목록Child
POST/habits/:id/check습관 완료 체크 → pending or completedChild
POST/habits/:id/approve부모 승인 → EXP/골드 지급Parent
GET/habits/:id/streak스트릭 히스토리Both
POST/party파티 생성Child
POST/party/join초대 코드로 파티 참여Child
POST/party/boss/start보스전 시작Child
WS/ws/party/:id보스전 실시간 동기화Child
GET/report/weekly주간 습관 리포트Parent
GET/report/children자녀 목록 + 요약 상태Parent
GET/leaderboard레벨/스트릭 랭킹 (Redis)Both

Monorepo 구조. apps/ 아래 백엔드와 프론트엔드 분리. packages/로 공유 로직.
