# E2E 테스트 체크리스트

Phase 6 완료를 위한 전체 플로우 테스트 및 검증 목록입니다.

## 📋 테스트 환경 설정

### 사전 준비
- [ ] 백엔드 API 서버 실행 중 (http://localhost:8000)
- [ ] PostgreSQL 데이터베이스 실행 중
- [ ] Redis 실행 중 (선택사항)
- [ ] 프론트엔드 개발 서버 실행 (http://localhost:3001)

### 테스트 데이터 준비
- [ ] 테스트용 부모 계정 존재
- [ ] 페어링 코드 생성됨
- [ ] 초기 보스 데이터 Seeded

## 🎯 핵심 사용자 플로우

### 1. 회원가입 플로우 ✅

#### 정상 케이스
- [ ] `/register` 페이지 접속
- [ ] 이메일 입력 (형식: test-child-[timestamp]@test.com)
- [ ] 비밀번호 입력 (8자 이상)
- [ ] 닉네임 입력 (2-20자)
- [ ] "회원가입" 버튼 클릭
- [ ] 자동 로그인 및 페어링 페이지로 이동
- [ ] 캐릭터 자동 생성 확인

#### 에러 케이스
- [ ] 빈 필드로 제출 시 검증 에러 표시
- [ ] 잘못된 이메일 형식 입력 시 에러
- [ ] 비밀번호 8자 미만 입력 시 에러
- [ ] 이미 존재하는 이메일로 가입 시 에러
- [ ] 네트워크 오류 시 에러 메시지 표시
